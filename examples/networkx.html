Dense Graphs using networkx & matplotlib

In this example we will create a simple fully-connected network of 10 nodes using networkX API.
The network will be rendered as an image (canvas) object using the default Matplotlib rendering engine.

import networkx as nx import matplotlib.pyplot as plt fig, ax = plt.subplots() G = nx.complete_graph(10) nx.draw(G, ax=ax) Element("networkx").write(fig)
